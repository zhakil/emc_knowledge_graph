欢迎使用EMC知识图谱桌面客户端

基于Electron的原生Windows桌面应用程序，提供完整的知识图谱构建和管理功能。

原生桌面体验
完全的Windows原生应用程序，支持系统托盘、文件关联、快捷键等桌面功能。
离线工作模式
支持本地文件处理和知识图谱构建，无需互联网连接即可使用核心功能。

🚀 准备构建Windows安装包

Electron配置已完成，可以使用以下命令构建Windows桌面应用程序：

npm run build && npm run dist

构建完成后将生成：

  • EMC知识图谱系统-1.0.0-x64.exe - Windows安装程序
  • 便携版目录 - 免安装版本
  • 自动更新包 - 增量更新文件
